Libros fantásticos Biblioteca virtual ¿Quiénes somos? Libros fantásticos... Y donde encontrarlos Somos un grupo de compañeros de la carrera que buscan realizar una biblioteca virtual para nuestro proyecto de tecnologías web. Nuestro proposito es realizar un aplicación de una biblioteca virtual para obtener distintos ejemplares de libros clásicos, modernos, juveniles, históricos, etc. Facilitar las operaciones presenciales de una biblioteca, de manera virtual.
